プライバシーポリシー

LunaCare（ルナケア） / 最終更新: 2026年4月26日

LunaCare（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、本ポリシーに従って情報を取り扱います。本アプリは ログイン機能を提供しておらず、氏名・メールアドレス・電話番号・住所・支払情報等の個人情報を取得しません。

1. 本アプリが取得しないもの

  • 氏名・メールアドレス・電話番号・住所
  • クレジットカード番号その他の決済情報
  • 位置情報（GPS）
  • 連絡先・カレンダー・写真ライブラリ
  • カメラ・マイクの直接取得情報
  • 広告識別子（IDFA）・解析サービス（Google Analytics 等）への送信
  • 第三者広告SDK経由の追跡情報

2. 端末内に保存される情報

感情記録・周期に関する設定・ルナモンプロフィール・チャット履歴・占い結果・各種アプリ設定は、ユーザーのiPhone内のローカル領域（SwiftData / UserDefaults / Keychain）にのみ保存されます。これらは、本ポリシー第3項に明記する場合を除き、外部のサーバーや第三者に送信されません。

ユーザーは、本アプリの設定 → データ → 「全データを削除」、または本アプリのアンインストールにより、保存されたすべての情報をいつでも削除できます。

3. AIサービス（OpenAI）への送信について

重要: AIケアチャット機能および一部のAI占い機能は、応答生成のため、ユーザーが入力したテキストを OpenAI, L.L.C.（米国所在）が提供するOpenAI API に送信します。本アプリの初回AI機能利用時に同意確認が表示されます。

送信される情報:

  • AIチャット欄に入力したテキスト
  • 応答品質向上のための匿名コンテキスト（現在の周期フェーズ名・最近の感情カテゴリ・星座・占いの種類）

送信されない情報:

  • 本ポリシー第1項に列挙したすべての情報（本アプリは元々取得しません）
  • 生理日・排卵日の具体的な日付
  • 感情記録の本文メモ（チャットに直接入力した内容を除く）
  • HealthKitから取得したデータ

OpenAIにおける取扱い: OpenAIに送信されたデータは、OpenAIのAPI利用ポリシーに従って処理されます。OpenAI APIは、デフォルトで送信されたデータをモデル学習に使用しません。データは不正利用検出のため最大30日間OpenAI側に保持された後、削除されます。

同意の撤回: ユーザーは、本アプリの設定からAIサービスの利用をいつでも停止できます。停止後は記録・周期・ルナモン育成等の他の機能は通常どおりご利用いただけます。

国際的なデータ転送: OpenAIの提供インフラは米国を含む複数の国に所在します。AI機能をご利用いただくことで、これらの国へのデータ転送に同意いただいたものとみなします。

4. HealthKit連携（任意）

ユーザーが許可した場合に限り、Apple HealthKit経由で月経記録を 読み取り、本アプリ内の周期予測精度向上に利用します。HealthKitから取得したデータは 本アプリの機能の範囲を超えて第三者に送信・共有されることはなく、広告・マーケティング目的にも一切使用されません。HealthKit連携はiOSの設定 → ヘルスケア、または本アプリの設定からいつでも解除できます。

5. 課金情報

プレミアムサブスクリプションの購入はApple Inc.のApp Storeを通じて処理され、本アプリは決済情報を取得・保存しません。本アプリが取得・保持するのは、購入されたプロダクトIDとサブスクリプションの有効期限の情報のみです。

6. パートナーシェア機能

パートナーシェア機能は、ユーザーが明示的に作成・確認したカード画像を、ユーザー自身がiOS純正の共有シートで選択した宛先に対してのみ送信します。本アプリのサーバーを経由せず、誰に・何を・いつ送信したかという情報を本アプリは記録しません。カード画像には生理日・排卵日・周期日数の具体的な数値・日付は含まれません。

7. 安全管理

本アプリは、合理的な範囲で技術的・組織的な安全管理措置を講じます。AIサービスとの通信はHTTPS/TLSで暗号化されています。ただし、インターネット通信および電子的保管の性質上、絶対的なセキュリティを保証するものではありません。

8. お問い合わせ

連絡先: 準備中（リリース時に正式な問い合わせ窓口を開設予定です）

本ポリシーに関するご質問・ご要望は、リリース後に開設する公式問い合わせ窓口までご連絡ください。なお、本アプリは端末内データのみを取り扱う性質上、確実なデータ削除は本アプリの設定 → データ → 「全データを削除」が即時かつ完全な手段となります。

9. ポリシーの変更

本ポリシーは、サービス内容や関連法令、Apple審査ガイドラインの更新に伴い改訂されることがあります。改訂後のポリシーは本ページに掲載された時点で効力を生じます。重要な変更がある場合は、本ページ上部の「最終更新」日付の更新と、合理的な範囲でアプリ内通知またはApp Store更新説明にてお知らせします。
